Detailed Description

Módulo com todas as funções lógicas e matemáticas.

Contém as funções para validar números, cálculos de matrizes, operações com vetores e o menu principal.

Author
Gabriel Salazar, Daniel Colavito, Leandro Apolo, Jorge Altamirano
Version
1.2
