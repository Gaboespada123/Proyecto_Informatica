Detailed Description

Programa principal para manipulação de vetores.

Este programa permite ao usuário inserir um vetor de 16 números e, em seguida, executar várias operações matemáticas e de manipulação de dados nesse vetor através de um menu interativo.

Author
Gabriel Salazar, Daniel Colavito, Leandro Apolo, Jorge Altamirano
Version
1.0
