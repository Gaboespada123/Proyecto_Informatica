O programa opera da seguinte maneira\+:


\begin{DoxyEnumerate}
\item {\bfseries{Inicialização\+:}} O usuário deve primeiro inserir 16 números inteiros (entre -\/10 e 27) para popular o vetor principal.
\item {\bfseries{Menu Interativo\+:}} Após a criação do vetor, um menu é exibido com 12 opções.
\item {\bfseries{Execução\+:}} O usuário pode selecionar qualquer opção (1-\/11) para realizar cálculos ou manipulações.
\item {\bfseries{Saída\+:}} A opção 12 encerra o programa.
