# Descrição do Fluxo de Trabalho

O programa opera da seguinte maneira:

1.  **Inicialização:** O usuário deve primeiro inserir 16 números inteiros (entre -10 e 27) para popular o vetor principal.
2.  **Menu Interativo:** Após a criação do vetor, um menu é exibido com 12 opções.
3.  **Execução:** O usuário pode selecionar qualquer opção (1-11) para realizar cálculos ou manipulações.
4.  **Saída:** A opção 12 encerra o programa.
