# Detalhes das Funções Principais

O programa divide-se em três categorias de funções:

### 1. Operações Matemáticas
* Cálculo do Seno
* Encontrar números primos
* Encontrar divisíveis por 3

### 2. Manipulação de Vetores
* Ordenar (decrescente)
* Misturar (metade-metade)
* Multiplicar pelo último elemento

### 3. Operações com Matrizes
* Criar matriz 2x16
* Criar matriz 16x16 (produto)
* Calcular determinante
